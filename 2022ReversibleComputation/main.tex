\subsection{Contributions}
\begin{itemize}
    \item Very strong confidence that \RPP is \PRF-complete, based on a fully detailed algorithmic translation.
    \item A simplified translation from \PRF to \RPP, as compared to the original one in \cite{DBLP:journals/tcs/PaoliniPR20} which is based on the formalization of \UPRF available in \LEAN library.
    \item Identification of a programming pattern common to interesting algorithms, one of which, the Cantor pairing, lies at the core of the translation from \PRF to \RPP.
    \item Concrete examples of reversible programming that, besides their formal certification, can be exported into a programming language as real and efficient as \CPP, as soon as \LEANFour will be delivered.
\end{itemize}
